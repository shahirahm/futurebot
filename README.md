# FutureBot

## Description
FutureBot is a **career guidance and talent connection platform** that connects **Students, Mentors, and Companies** in a single system.  
The goal of FutureBot is to help students choose the right career path, get expert guidance, and connect with industry opportunities.



## Project Overview

FutureBot bridges the gap between **education and industry** by providing:
- Career guidance for students
- Mentorship from experienced professionals
- Talent discovery for companies

The system is built using **role-based access control**, where each user role has its own dashboard and features.



##  User Roles

###  Student
Students use the platform to explore careers and grow skills.

**Features:**
- Student registration and login
- Career recommendations
- View and connect with mentors
- Track skill development
- Access guidance and resources


###  Mentor
Mentors guide students using their experience.

**Features:**
- Mentor registration and login
- Manage mentor profile
- View student requests
- Provide career advice
- Share learning resources

---

### Company
Companies find skilled and potential candidates.

**Features:**
- Company registration and login
- Post job and internship opportunities
- View student profiles
- Shortlist candidates
- Connect with mentors and students

---

## Authentication System

## Installation & Setup

1. Clone the repository:
   git clone https://github.com/your-username/futurebot.git
2. Move project to XAMPP htdocs:
   C:/xampp/htdocs/futurebot
3. Create a database:
   Open phpMyAdmin
   Create database futurebot
   Import the .sql file
4. Configure database connection:
   Update db.php with database credentials
5. Run the project:
   http://localhost/futurebot
